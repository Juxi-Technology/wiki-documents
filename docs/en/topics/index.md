# Tech Topics

Welcome to Juxi Technology Tech Topics Knowledge Base. This resource helps you master the robot learning and automation technologies we focus on.

- [**Robot Learning**](/en/topics/robot-learning/) - Full-stack robot learning based on LeRobot
  - Deep dive into robot learning algorithms: ACT, Diffusion Policy, Pi0, etc.
  - Complete workflow for data collection, training, and deployment
  - Learning path from beginner to master

- [**Arm & Vision**](/en/topics/arm-vision/) - Deep integration of robotic arms and computer vision
  - Hand-eye calibration, visual servo control
  - Object detection and pose estimation
  - 3D vision and depth camera applications

- [**Open Source Projects**](/en/topics/open-source/) - In-depth analysis and secondary development of open-source projects
  - Source code analysis and architecture design of LeRobot
  - MuJoCo and IsaacGym simulation environments
  - HuggingFace ecosystem applications

Overall, these tech topics will help you deeply learn and practice in robotics, covering the complete technical stack from basic to advanced.
