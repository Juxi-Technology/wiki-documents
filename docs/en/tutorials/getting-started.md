
# Getting Started

Welcome to Juxi Technology products! This tutorial will help you get started quickly.

## Prerequisites

Before starting, make sure you have the following:

- Product hardware
- USB cable
- Computer (Windows/macOS/Linux)

## Step 1: Connect Hardware

1. Connect the product to your computer using the USB cable
2. Wait for device drivers to automatically install
3. Confirm the device is recognized correctly

## Step 2: Install Software

```bash
# Install dependencies
npm install

# Start application
npm start
```

## Step 3: Start Using

You can now start using the product! Check out more tutorials for detailed features.
