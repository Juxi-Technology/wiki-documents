
# Hardware Setup

This chapter details the product hardware connection method.

## Interface Description

The product includes the following interfaces:

- USB-C: Power and data transmission
- GPIO: General-purpose input/output interface
- HDMI: Video output

## Connection Steps

1. **Connect Power**: Use USB-C cable to connect power
2. **Connect Peripherals**: Connect other peripherals as needed
3. **Power On**: Press power button to start the device

## Notes

- Ensure the power voltage meets requirements
- Avoid plugging/unplugging GPIO devices while powered on
