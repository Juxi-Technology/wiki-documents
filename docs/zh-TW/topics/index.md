# 技術專題

歡迎來到鉅犀科技技術專題知識庫。此資源旨在幫助你掌握我們專注的機器人學習與自動化技術。

- [**機器人學習**](/zh-TW/topics/robot-learning/) - 基於 LeRobot 的全棧機器人學習技術
  - 深入探索機器人學習算法：ACT、Diffusion Policy、Pi0 等
  - 數據採集、訓練、部署的完整流程
  - 從入門到精通的學習路徑

- [**機械臂與視覺**](/zh-TW/topics/arm-vision/) - 機械臂與計算機視覺的深度集成
  - 手眼標定、視覺伺服控制
  - 物體檢測與位姿估計
  - 3D 視覺與深度相機應用

- [**開源項目**](/zh-TW/topics/open-source/) - 開源項目深度解析與二次開發
  - LeRobot 源碼解析與架構設計
  - MuJoCo 與 IsaacGym 模擬環境
  - HuggingFace 生態系統應用

總體而言，這些技術專題將幫助你在機器人領域進行深入學習和實踐，從基礎到進階，覆蓋完整的技術棧。
