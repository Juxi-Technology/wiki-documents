# 贡献者社区

欢迎加入钜犀科技贡献者社区！我们诚邀充满热情的创客、开发者和工程师与我们携手合作。

---

## 为什么加入？

| 提升影响力 | 技术成长 | 社区人脉 |
|---|---|---|
| 你的贡献将帮助全球用户学习机器人技术 | 与核心团队交流，获得技术指导和反馈 | 结识志同道合的朋友，拓展专业人脉 |

---

## 如何贡献

### 1. 撰写教程

分享你使用 SO-ARM101 的经验，撰写教程帮助他人。

**如何开始：**
- 选择一个主题（应用场景、技术解析等）
- 参考我们的[教程模板](/tutorials/so-arm101/template-tutorial)
- 提交 Pull Request

### 2. 翻译文档

帮助我们将文档翻译成更多语言，让更多用户受益。

**现有语言：**
- 简体中文
- 英文
- 繁体中文

**翻译指南：** [如何参与翻译](/community/translation-guide)

### 3. 提交案例

分享你的使用案例，展示你的项目成果。

**案例类型：**
- 教育应用
- 研究项目
- 商业应用

### 4. 报告问题

发现 Bug 或有改进建议？提交 Issue 帮助我们改进。

### 5. 代码贡献

对相关开源项目的代码贡献，包括 LeRobot 等。
